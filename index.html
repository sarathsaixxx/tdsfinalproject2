Get Started in Minutes

Follow these instructions to set up and run the TDS Data Analyst Agent on your local machine or deploy it to production.

1. Install Dependencies

pip install -r requirements.txt

2. Configure Environment

Create a .env file in the project root with your API key:

gemini_api_1=your_google_generative_ai_api_key
LLM_TIMEOUT_SECONDS=150

3. Launch the App

python app.py

For production environments, we recommend using a robust server like Gunicorn:

gunicorn app:app -w 4 -k uvicorn.workers.UvicornWorker

Use the provided Dockerfile to build and run the application in a container:

# Build the Docker image
docker build -t tds-agent .

# Run the container
docker run -p 8000:8000 -v $(pwd)/.env:/app/.env tds-agent
